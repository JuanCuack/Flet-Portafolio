**Flet Portafolio** es una solución de software interactiva desarrollada para la asignatura de **Programación II** en la **Universidad de Cundinamarca**. El repositorio centraliza mi trayectoria académica y técnica mediante una interfaz web moderna, demostrando la potencia de Python para crear experiencias de usuario (UX) fluidas y profesionales.

Este proyecto aplica conceptos avanzados de **Programación Orientada a Objetos (POO)** y gestión de estados asíncronos, todo operando bajo un entorno virtual controlado para garantizar la portabilidad y estabilidad de las dependencias.

---

## 💎 Características del Portafolio (Key Features)

* **🎨 Interfaz Dinámica y Responsiva:** Sistema visual construido con Flet que se adapta a diferentes resoluciones, integrando un diseño limpio y profesional.
* **🌗 Gestión de Temas:** Implementación de cambio de modo claro/oscuro (Light/Dark mode) para mejorar la accesibilidad y experiencia de usuario.
* **🏗️ Arquitectura Modular:** Organización del código basada en clases y componentes independientes, facilitando el mantenimiento y la escalabilidad del software.
* **🖼️ Integración de Assets Personalizados:** Manejo eficiente de recursos externos, incluyendo tipografías técnicas (como *Starjhol*), iconos vectoriales (SVG) y gráficos de alta calidad.

---

## 🛠️ Especificaciones Técnicas

* **Lenguaje:** Python 3.13
* **Framework Core:** Flet (UI as Code).
* **Gestión de Entorno:** Python Virtual Environment (`.venv`).
* **Dependencias:** Gestionadas mediante `requirements.txt` para replicabilidad.
* **Despliegue:** Optimizado para ejecución en navegador web (`WEB_BROWSER`).

## 🚀 Implementación Rápida

1. **Clona el repositorio:**
    ```bash
    git clone [https://github.com/JuanCuack/Flet-Portafolio.git](https://github.com/JuanCuack/Flet-Portafolio.git)
    cd Flet-Portafolio
    ```

2. **Configuración del Ambiente:**
    ```text
    python -m venv .venv

    # Activar en Windows:
    .\.venv\Scripts\activate

    # Instalar dependencias:
    pip install -r requirements.txt
    ```

3. **Organización del Proyecto:**
    ```text
    Flet-Portafolio/
    ├── .venv/ (Excluido vía .gitignore)
    ├── assets/ (Imágenes, fuentes y logos)
    ├── main.py (Lógica principal de la App)
    ├── requirements.txt
    ├── .gitignore
    └── README.md
    ```

4. **Instrucciones de Ejecución:**
    ```bash
    python main.py
    ```
